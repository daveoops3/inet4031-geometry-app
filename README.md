# inet4031-geometry-app
## Purpose
This website calculates the volume of a cylinder when given the height and radius or the volume and surface area of a sphere when given the radius.

## about
To access the website you must put the ip address of the VM with a port 5000.
Then when the website loads you are give a selection of cylinder or sphere, select the one you want to calculate the volume of.
Once you selected the one you want click submit.
In the next page enter the radius and height if asked then hit submit
Once you hit submit it will tell you the volume and surface area
